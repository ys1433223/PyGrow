# PyGrow 开发任务拆分

## 第一阶段：MVP 必做功能

目标：跑通主流程。

主流程：

```text
注册登录
    ↓
能力测评
    ↓
推荐课程
    ↓
课程学习
    ↓
完成练习
    ↓
获得积分
    ↓
查看学习报告
任务1：初始化项目
要求：
1.	创建 Vue3 + Vite 前端项目；
2.	创建 FastAPI 后端项目；
3.	配置 MySQL；
4.	配置 Docker Compose；
5.	配置 Nginx；
6.	创建基础 README。
任务2：用户系统
要求：
1.	注册；
2.	登录；
3.	JWT鉴权；
4.	个人资料查看；
5.	个人资料修改。
任务3：课程模块
要求：
1.	课程列表；
2.	课程详情；
3.	视频播放；
4.	学习进度记录。
任务4：能力测评
要求：
1.	测评题目展示；
2.	答题提交；
3.	自动评分；
4.	等级判断；
5.	推荐课程。
任务5：练习模块
要求：
1.	题库；
2.	每日一练；
3.	提交答案；
4.	自动判题；
5.	错题记录。
任务6：积分等级系统
要求：
1.	完成课程获得经验；
2.	完成练习获得经验；
3.	根据经验值计算段位；
4.	首页展示段位和经验条。
任务7：学习报告基础版
要求：
1.	统计课程完成数；
2.	统计练习数量；
3.	统计正确率；
4.	展示当前等级；
5.	展示薄弱知识点。
________________________________________
第二阶段：增强功能
任务8：AI代码导师
要求：
1.	接入大模型 API；
2.	实现学习问答；
3.	实现代码解释；
4.	实现代码纠错；
5.	保存 AI 对话记录。
任务9：在线代码运行
要求：
1.	前端代码编辑器；
2.	后端运行 Python 代码；
3.	返回 stdout；
4.	返回 stderr；
5.	限制执行时间；
6.	预留 Docker 沙箱升级结构。
任务10：项目挑战中心
要求：
1.	项目列表；
2.	项目详情；
3.	项目提交；
4.	项目完成记录；
5.	项目奖励经验。
任务11：错题本
要求：
1.	自动记录错题；
2.	错题重练；
3.	按知识点筛选错题。
任务12：时间点笔记
要求：
1.	视频播放时添加笔记；
2.	自动记录视频时间；
3.	点击笔记跳转时间点。
________________________________________
第三阶段：扩展功能
任务13：学习社区
要求：
1.	发帖；
2.	评论；
3.	点赞；
4.	收藏；
5.	内容长度限制；
6.	违规词过滤。
任务14：徽章系统
要求：
1.	创建徽章；
2.	自动判断徽章条件；
3.	展示用户徽章。
任务15：排行榜
要求：
1.	按经验值排行；
2.	按项目完成数排行；
3.	按连续学习天数排行。
任务16：后台管理
要求：
1.	用户管理；
2.	课程管理；
3.	题库管理；
4.	项目库管理；
5.	帖子管理。
________________________________________
第四阶段：未来扩展
任务17：编程农场
要求：
1.	完成学习任务获得种子；
2.	完成练习获得金币；
3.	完成项目解锁作物；
4.	农场等级与用户等级联动。
任务18：任务地图
要求：
1.	将课程路径可视化为地图；
2.	每个知识点是一个关卡；
3.	完成关卡解锁下一关。
任务19：宠物养成
要求：
1.	学习获得宠物经验；
2.	连续学习提高宠物成长值；
3.	项目挑战获得宠物道具。
