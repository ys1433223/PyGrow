# PyGrow 后端接口设计

接口基础路径：

```text
/api

统一返回格式：

{
  "code": 200,
  "message": "success",
  "data": {}
}
一、用户接口
POST /api/auth/register
POST /api/auth/login
GET  /api/user/profile
PUT  /api/user/profile
二、首页接口
GET /api/home/dashboard
GET /api/home/recommendations
GET /api/home/daily-tasks
三、课程接口
GET  /api/courses
GET  /api/courses/{id}
POST /api/courses/{id}/progress
GET  /api/courses/{id}/notes
POST /api/courses/{id}/notes
四、能力测评接口
GET  /api/assessment/questions
POST /api/assessment/submit
GET  /api/assessment/result

测评提交后返回：

{
  "level": "初级",
  "score": 65,
  "weak_points": ["循环结构", "列表"],
  "recommended_courses": []
}
五、练习接口
GET  /api/practice/daily
GET  /api/practice/by-knowledge
POST /api/practice/submit
GET  /api/practice/wrong-questions
六、代码运行接口
POST /api/code/run
POST /api/code/submit

代码运行接口输入：

{
  "code": "print('hello')"
}

代码运行接口输出：

{
  "success": true,
  "stdout": "hello",
  "stderr": ""
}
七、项目挑战接口
GET  /api/projects
GET  /api/projects/{id}
POST /api/projects/{id}/submit
GET  /api/projects/my-submissions
八、AI导师接口
POST /api/ai/explain-code
POST /api/ai/debug-code
POST /api/ai/ask
POST /api/ai/study-suggestion

AI导师接口输入：

{
  "question": "为什么我的for循环报错？",
  "code": "for i in range(10)\nprint(i)",
  "knowledge_point": "for循环",
  "level": "入门级"
}

AI导师接口输出：

{
  "answer": "你的代码第一行缺少冒号，并且第二行需要缩进。"
}
九、学习报告接口
GET /api/report/summary
GET /api/report/knowledge-points
GET /api/report/progress
十、游戏化接口
GET  /api/gamification/level
GET  /api/gamification/badges
GET  /api/gamification/ranking
POST /api/gamification/complete-task
十一、社区接口
GET  /api/posts
POST /api/posts
GET  /api/posts/{id}
POST /api/posts/{id}/comments
POST /api/posts/{id}/like
十二、后台管理接口
GET    /api/admin/users
GET    /api/admin/courses

PUT    /api/admin/courses/{id}
DELETE /api/admin/courses/{id}

GET    /api/admin/questions
POST   /api/admin/questions
PUT    /api/admin/questions/{id}
DELETE /api/admin/questions/{id}

GET    /api/admin/projects
POST   /api/admin/projects

DELETE /api/admin/projects/{id}
