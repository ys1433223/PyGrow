# PyGrow：AI Python 自主学习平台项目总说明

## 一、项目名称

PyGrow：AI Python 自主学习平台

## 二、项目定位

本项目是一款面向大学本科学生的 Python 自主学习平台，重点解决传统编程学习过程中学习路径不清晰、实践项目不足、代码问题难以及时解决、学习动力不足等问题。

平台以“能力测评—分级学习—课程学习—在线练习—项目挑战—AI导师—成长激励”为核心学习路径，帮助学习者从 Python 入门逐步成长为能够独立完成项目的学习者。

## 三、目标用户

大学本科学生，尤其是教育技术学、信息技术、计算机基础课程相关专业或对 Python 编程感兴趣的学生。

## 四、核心价值

平台不是普通的视频课程网站，而是一个集课程学习、在线练习、AI辅导、项目实践和游戏化成长于一体的 Python 学习支持系统。

## 五、核心问题

当前大学生在 Python 自主学习过程中主要存在以下问题：

1. 学习路径不清晰，不知道从哪里开始、下一步学什么；
2. 理论学习与项目实践脱节；
3. 遇到代码报错时难以及时获得帮助；
4. 缺少持续学习动力，容易中途放弃；
5. 学习资源分散，需要频繁切换多个平台。

## 六、核心功能

平台主要包括：

1. 用户注册登录；
2. 能力测评与自动定级；
3. Python 课程学习；
4. 在线练习与自动判题；
5. 在线 Python 编程环境；
6. AI 代码导师；
7. 项目挑战中心；
8. 学习报告；
9. 游戏化成长体系；
10. 学习社区；
11. 后台管理。

## 七、推荐技术栈

前端：Vue3 + Vite + Element Plus  
后端：FastAPI  
数据库：MySQL  
部署：Docker Compose + Nginx  
AI接口：DeepSeek / 通义千问 / 智谱 / OpenAI API  
在线代码运行：MVP 阶段可用 subprocess，后续升级为 Docker 沙箱。
