# PyGrow 功能模块说明

## 一、首页模块

### 功能目标

让用户快速了解自己的学习状态、当前等级、学习进度和今日任务。

### 主要功能

1. 当前等级展示；
2. 经验值进度条；
3. 继续学习入口；
4. 个性化推荐课程；
5. 每日任务；
6. 错题本入口；
7. 学习报告入口；
8. 能力测评入口。

---

## 二、用户模块

### 功能目标

完成用户注册、登录、个人信息管理和学习身份初始化。

### 主要功能

1. 用户注册；
2. 用户登录；
3. 退出登录；
4. 修改个人资料；
5. 查看个人主页；
6. 新手引导。

---

## 三、能力测评模块

### 功能目标

通过测评判断用户当前 Python 学习水平，并推荐合适的学习起点。

### 能力等级

1. 入门级；
2. 初级；
3. 中级；
4. 高级。

也可以映射为游戏化段位：

1. 黑铁；
2. 白银；
3. 黄金；
4. 铂金；
5. 钻石；
6. 星耀；
7. 王者；
8. 传奇。

### 测评内容

1. Python基础语法；
2. 变量与数据类型；
3. 条件语句；
4. 循环结构；
5. 列表与字典；
6. 函数；
7. 文件操作；
8. 简单项目理解。

### 结果规则

```text
0% - 40%：入门级
41% - 60%：初级
61% - 80%：中级
81% - 100%：高级
输出内容
1.	当前能力等级；
2.	薄弱知识点；
3.	推荐学习路径；
4.	推荐课程；
5.	推荐练习。

四、课程中心模块
功能目标
为学生提供系统化 Python 课程内容。
课程结构
根据能力等级推荐资源但是所有用户使用同一套资源
Python入门
├── 认识Python
├── print输出
├── input输入
├── 变量
└── 数据类型

Python初级
├── 条件语句
├── 循环结构
├── 列表
├── 字典
└── 字符串处理

Python中级
├── 函数
├── 模块
├── 文件操作
├── 异常处理
└── 简单数据处理

Python高级
├── 面向对象
├── 爬虫基础
├── 数据分析
├── 可视化
└── 综合项目
单个课程页面内容
1.	课程视频；
2.	课程简介；
3.	知识点标签；
4.	课程资源包；
5.	时间点笔记；
6.	相关练习；
7.	相关项目。
________________________________________
五、时间点笔记模块
功能目标
支持学生在观看视频时记录笔记，方便复习。
功能设计
1.	用户在视频播放过程中点击“添加笔记”；
2.	系统自动记录当前视频时间点；
3.	用户输入笔记内容；
4.	后续点击笔记可跳转到对应视频时间点。
________________________________________

六、练习中心模块
功能目标
通过题库练习帮助学生巩固知识点。
练习类型
1.	单选题；
2.	多选题；
3.	判断题；
4.	填空题；
5.	代码题；
6.	简答题。
练习模块
1.	每课一练；
2.	每日一练；
3.	专项练习；
4.	错题本；
5.	晋级测试。
________________________________________
七、在线编程模块
功能目标
让用户无需本地配置环境即可在线运行 Python 代码。
基础功能
1.	代码编辑器；
2.	运行代码；
3.	显示输出；
4.	显示错误信息；
5.	保存代码；
6.	提交代码。
技术建议
前端使用 Monaco Editor 或 CodeMirror。
MVP 阶段可以使用后端 Python subprocess 运行代码，但必须限制运行时间。
正式部署建议使用 Docker 沙箱。
________________________________________
八、项目挑战中心模块
功能目标
通过项目驱动学习，提高学生实践能力。
项目等级
1.	入门项目；
2.	初级项目；
3.	中级项目；
4.	高级项目。
项目示例
入门项目：
1.	猜数字小游戏；
2.	BMI计算器；
3.	随机点名器。
初级项目：
1.	学生成绩管理系统；
2.	通讯录系统；
3.	简易计算器。
中级项目：
1.	图书管理系统；
2.	文件批量整理工具；
3.	数据可视化小工具。
高级项目：
1.	校园数据分析项目；
2.	网页爬虫项目；
3.	AI聊天助手项目。
________________________________________
九、AI代码导师模块
功能目标
为学生提供即时、个性化的编程学习支持。
功能类型
1.	代码解释；
2.	代码纠错；
3.	学习答疑；
4.	项目提示；
5.	学习建议。
AI导师原则
AI导师不应只给答案，而应帮助学生理解原因，提供逐步提示。

十一、游戏化成长模块
功能目标
通过积分、等级、徽章和任务机制提升学习动力。
成长元素
1.	经验值；
2.	积分；
3.	段位；
4.	徽章；
5.	排行榜；
6.	每日任务；
7.	连续学习天数。
积分规则示例
完成一个课程视频：+10经验
完成每日一练：+15经验
完成一次代码题：+20经验
完成一个项目挑战：+50经验
通过晋级测试：+100经验
发表高质量帖子：+10经验
连续学习7天：获得“坚持学习者”徽章
________________________________________
十二、学习社区模块
功能目标
为学习者提供交流、提问和项目展示空间。
社区分区
1.	问答区；
2.	技术分享区；
3.	项目展示区；
4.	课程评价区。
发帖限制
1.	标题不能为空；
2.	内容不少于10个字；
3.	禁止违规词；
4.	同一用户短时间内不能频繁发帖；
5.	支持管理员删除违规内容。
________________________________________
十三、后台管理模块
功能目标
方便管理员维护平台内容。
管理功能
1.	用户管理；
2.	课程管理；
3.	题库管理；
4.	项目库管理；
5.	社区内容管理；
6.	评论管理；
7.	数据统计。
